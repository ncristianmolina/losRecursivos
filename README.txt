///////////////////////////////TA-TE-TI//////////////////////////////

El proyecto está compuesto por las siguientes librerías: 

* inicio-sesion.c (Autor Cristian Molina)
* mock.c (Autora Luciana B Hamade)
* partidas.c (Autora Luciana B Hamade)
* partidasXjugador.c (Autora Luciana B Hamade)
* ranking.c (Autor Cristian Molina)
* registro.c (Autor Cristian Molina)
* tateti.c (Autores Luciana B Hamade y Cristian Molina)

-----------------------------------------------------------------------------------------------------------------------------------------------------

inicio-sesión: contiene las funciones de login con sus respectivas validaciones.
CRUD de jugadores

----------------------------------------------------------------------------------------------------------------------------------------------------
mock: crea jugadores de manera aleatoria. Permite la lectura del archivo jugadores.dat
----------------------------------------------------------------------------------------------------------------------------------------------------
partidas : crea partidas de manera aleatoria. Permite la lectura del archivo partidas.dat
----------------------------------------------------------------------------------------------------------------------------------------------------
partidasXjugador: crea partidas por jugador de manera aleatoria. Permite la lectura del archivo partidasXjugador.dat
----------------------------------------------------------------------------------------------------------------------------------------------------
ranking: lee archivo de partidas por jugador, calcula total de partidas y realiza los cálculos para obtener las estadísticas de manera porcentual
----------------------------------------------------------------------------------------------------------------------------------------------------
registro: genera los registros de los jugadores de manera manual. 

---------------------------------------------------------------------------------------------------------------------------------------------------
tateti: maneja la lógica del juego
